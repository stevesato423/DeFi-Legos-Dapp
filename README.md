# defi-legos

Simple Dapp that allows to create single transactions composed of multiple interactions with the main DeFi  protocols.

&nbsp;

***

&nbsp;

### :rocket: How to start

```
yarn install
```

Then create an __.env__ file with the following parameters:

```
REACT_APP_INFURA_ENDPOINT=
REACT_APP_INFURA_ID=
```

and then:

```
yarn start
```
