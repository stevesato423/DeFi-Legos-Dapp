# defi-legos

Simple dashboard for interacting with Defi protocols through drag and drop

&nbsp;

***

&nbsp;

### :rocket: How to start

```
yarn install
```

Then create an __.env__ file with the following parameters:

```
REACT_APP_INFURA_ENDPOINT=
REACT_APP_INFURA_ID=
```

and then:

```
yarn start
```
